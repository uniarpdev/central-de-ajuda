# Diário de Classe e Plano de Aula

Nesta tela, o professor faz o registro pedagógico de cada encontro realizado.

## 📝 Registro de Aula
Para cada dia de aula na grade horária:
1.  Selecione a disciplina e a turma.
2.  Descreva o conteúdo ministrado conforme o plano de ensino.
3.  Informe a data e o horário da aula.
4.  Clique em **Salvar Registro**.

## 📄 Plano de Ensino
Visualize o plano de ensino completo da disciplina para garantir o cumprimento das ementas e objetivos de aprendizagem. O plano de ensino aprovado pela coordenação servirá de base para seus lançamentos de conteúdo.
