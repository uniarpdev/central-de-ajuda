# Configuração de Ambiente

Após o login, esta é a tela mais importante para garantir que você veja as informações corretas.

## 🛠️ O que configurar?

1.  **Papel:** Escolha como deseja atuar (ex: Aluno, Professor, Coordenador).
2.  **Nível de Ensino:** Filtre o nível de ensino (ex: Graduação, Pós-Graduação, Colégio).
3.  **Período Letivo:** Selecione o semestre atual (ex: 2024/1).

## 🔄 Troca Rápida
Você pode alterar essas configurações a qualquer momento clicando no atalho **Ambiente** no menu superior direito. Isso permite alternar entre papéis sem precisar sair do sistema.
