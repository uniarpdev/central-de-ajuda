# Criar Novo Plano de Compensação

Nesta tela, você deve definir como irá compensar seu saldo devedor ou credor de horas.

## 📝 Regras de Preenchimento
Para que o sistema permita o envio para aprovação, você deve seguir estas diretrizes:

1.  **Saldo Total:** 100% das horas do saldo devem ser distribuídas no cronograma.
2.  **Limite Diário:** Você pode planejar no máximo **2 horas extras** por dia.
3.  **Jornada Máxima:** A soma da sua jornada normal + a compensação não pode ultrapassar **10 horas totais** no dia.
4.  **Choque de Horários:** O sistema não permite planejar compensação em horários que coincidam com sua grade de trabalho atual ou aulas (se for professor).

## 🚀 Envio para Aprovação
Após preencher todo o cronograma e o saldo restante for zero, o botão **Submeter para Aprovação** ficará disponível.
*   **Status Pendente:** Assim que submetido, seu gestor imediato será notificado para análise.
