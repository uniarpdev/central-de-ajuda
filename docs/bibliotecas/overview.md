# Bibliotecas e APIs

Para garantir a consistência e agilidade no desenvolvimento, utilizamos um conjunto de bibliotecas e APIs compartilhadas.

## APIs de Integração
- **[API TOTVS](https://github.com/uniarpdev/lib-api-totvs):** Cliente de integração com ERP RM.
- **[API LDAP](https://github.com/uniarpdev/lib-api-ldap):** Comunicação com Active Directory.
- **[API Grupo A](https://github.com/uniarpdev/lib-api-grupoa):** Conector para o ecossistema +Campus.
- **[API Valorizza](https://github.com/uniarpdev/lib-api-valorizza):** Conector para gestão de carreira.

## Helpers
- **[Dotnet Utils](https://github.com/uniarpdev/lib-dotnet-utils):** Utilitários gerais para C#.
- **[Uniarp Tasks](https://github.com/uniarpdev/uniarpTasks):** Automações e tarefas agendadas.
